# Django Kubes

A template project used to test Kubernetes and Docker with Django, Quart, Vite, Celery and Go. This project uses:
* Redis
* RabbitMQ
* PostGres

The websites a served using Traefik.

## Frontend

A simple Vite project that includes Bootstrap and can optionally be rendered using Tailwindcss (PrimeVue or Shad-cn).
